### Backend development 
1. Create a git repo 
    git init 
2. Add .gitignore file 
3.create a .env file for environment variables
    -Read data from .env with "dotenv" module
    -npm install dotenv
4. Generate package.json 
    npm init -y
    npm install express
    type:module
    main:server.js
5. Create a Express app
    npm install express
6. connect to database 
    npm install mongoose
7.Add Middlewares (body parser, error handlers)
8.DEsign Schemas and create models
9.Design REST APIs for all resources
10. To create a resuable code for both author and user as well as for login for admin too we need to create a separate service to reuse
